# Dashboard 控制台页面设计文档

> **页面路径**: `/`  
> **组件文件**: `pages/Dashboard.vue`  
> **状态**: Active

---

## 1. 概述

### 1.1 目标
Dashboard 是用户登录后的首屏，提供系统概览、市场数据、快速操作入口，让用户一眼掌握全局状态。

### 1.2 用户故事
- 作为交易员，我想快速查看市场指数概况，以便了解当日市场环境
- 作为研究员，我想一键进入常用功能模块，减少导航成本
- 作为管理员，我想确认系统 API 连接状态和数据更新情况

---

## 2. 信息架构

### 2.1 页面结构

```
Dashboard 页面
├── 页面标题区
│   └── "控制台" + 副标题
├── 导航磁贴区 (NavTiles)
│   ├── 回测引擎
│   ├── 因子选股
│   ├── 策略实验室
│   ├── AI Copilot
│   └── 批量回测
├── 市场数据区 (MarketMetrics)
│   ├── 指数选择器
│   ├── 日期选择器
│   ├── 刷新按钮
│   └── 指标卡片 (4列)
│       ├── 收盘价
│       ├── 涨跌幅
│       ├── 成交量
│       └── 成交额
└── 快速回测区 (QuickBacktest)
    ├── 简化表单
    └── 结果预览
```

### 2.2 数据流

```
用户进入 Dashboard
    ↓
自动加载市场数据 (默认上证指数 + 最新交易日)
    ↓
展示导航磁贴 + 市场指标 + 快速回测表单
    ↓
用户交互 (切换指数/日期/运行快速回测)
    ↓
更新对应区域数据
```

## 4. 组件规范

### 4.1 导航磁贴 (NavTiles)

**布局**: CSS Grid, `repeat(auto-fill, minmax(220px, 1fr))`

**磁贴样式**:
- 背景: `var(--q-surface)`
- 圆角: `var(--q-radius)`
- 内边距: `20px`
- Hover: `translateY(-2px)` + 阴影增强

**磁贴内容**:
- 顶部: 图标 (40px, 主题色)
- 中部: 标题 (16px, bold) + 描述 (13px, `var(--q-text2)`)
- 底部: 箭头图标 (暗示可点击)

### 4.2 市场指标卡片

**布局**: 4列等宽网格

**卡片内容**:
| 指标 | 格式 | 颜色规则 |
|------|------|----------|
| 收盘价 | `3,500.00` | 无 |
| 涨跌幅 | `+0.50%` / `-0.30%` | 正绿负红 |
| 成交量 | `2.1B` / `1.5M` | 无 |
| 成交额 | `¥300.5B` | 无 |

### 4.3 快速回测表单

**布局**: Inline 表单，一行展示

**字段**:
| 字段 | 组件 | 宽度 |
|------|------|------|
| 策略 | NSelect | 140px |
| 股票代码 | NInput | 120px |
| 开始日期 | NDatePicker | 140px |
| 结束日期 | NDatePicker | 140px |
| 运行按钮 | NButton (primary) | auto |

## 5. 交互设计

### 5.1 加载状态

| 场景 | 反馈 |
|------|------|
| 页面初始加载 | 骨架屏 (Skeleton) |
| 切换指数 | 指标卡片 loading |
| 运行快速回测 | 按钮 loading + 结果区域 skeleton |

### 5.2 空状态

- 市场数据加载失败: 显示错误提示 + 重试按钮
- 快速回测无结果: 显示"未找到数据"提示

### 5.3 错误处理

- API 断开: 顶部显示警告横幅
- 数据加载失败: 卡片内显示错误图标 + 重试

---

## 6. 视觉规范

### 6.1 色彩

- 页面背景: `var(--q-bg)`
- 卡片背景: `var(--q-surface)`
- 磁贴主题色: 每个磁贴不同 (蓝/紫/橙/绿/红)
- 涨跌幅: `var(--q-success)` / `var(--q-danger)`

### 6.2 字体

- 页面标题: 24px, bold
- 卡片标题: 16px, bold
- 指标数值: 20px, bold, mono
- 描述文字: 13px, `var(--q-text2)`

## 8. 性能优化

- 市场数据缓存 5 分钟
- 磁贴图标使用 `markRaw()`
- 快速回测结果使用 `shallowRef()`
