# Copilot 策略 Copilot 页面设计文档

> **页面路径**: `/copilot`  
> **组件文件**: `pages/Copilot.vue`  
> **状态**: Active

---

## 1. 概述

### 1.1 目标
AI 驱动的策略生成助手，通过自然语言对话帮助用户创建、优化和解释量化交易策略。

### 1.2 用户故事
- 作为新手用户，我想用自然语言描述策略思路，让 AI 帮我生成代码
- 作为研究员，我想让 AI 解释某个策略的底层逻辑
- 作为交易员，我想让 AI 优化现有策略的参数

---

## 2. 信息架构

### 2.1 页面结构

```
Copilot 页面
├── 聊天界面
│   ├── 消息列表 (用户 / AI)
│   ├── 输入框
│   └── 快捷提示
└── 策略预览区 (可选)
    ├── 生成的策略代码
    └── 参数配置
```

## 4. 交互设计

### 4.1 用户流程

```
[进入页面]
    ↓
[显示欢迎消息 + 快捷提示]
    ↓
[用户输入策略描述]
    ↓
[AI 生成策略代码 + 解释]
    ↓
[用户可: 继续对话 / 保存策略 / 直接回测]
```

### 4.2 边界状态

| 状态 | 触发条件 | UI 表现 |
|------|---------|---------|
| 首次进入 | 无历史对话 | 显示欢迎消息 + 快捷提示 |
| AI 思考中 | 用户发送消息后 | 显示 typing 动画 |
| 代码生成 | AI 返回代码 | 代码块支持复制 + 高亮 |
| 错误 | API 失败 | 显示错误提示 + 重试 |

---

## 5. 视觉规范

### 5.1 消息气泡

- 用户消息: 右对齐, `var(--q-primary)` 背景
- AI 消息: 左对齐, `var(--q-surface)` 背景
- 代码块: `var(--q-bg)` 背景 + 语法高亮

### 5.2 快捷提示

- 布局: 水平排列的 tag 按钮
- 样式: `var(--q-surface2)` 背景, hover 变亮

---

## 6. API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/copilot/chat` | POST | 发送对话消息 |
| `/api/copilot/strategy` | POST | 生成策略代码 |

---

## 7. 性能优化

- 消息列表使用虚拟滚动（>50 条时）
- 代码高亮使用 Web Worker
- 对话历史本地缓存
